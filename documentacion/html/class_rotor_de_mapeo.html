Descripción detallada

Disco de cifrado César implementado como lista circular.

Contiene el alfabeto A-Z en una lista circular doblemente enlazada. La cabeza indica la posición cero actual. Permite rotar para cambiar el mapeo de caracteres de forma dinámica.
